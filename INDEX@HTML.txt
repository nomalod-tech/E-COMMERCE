## 2?? functions.php (WooCommerce + Theme Setup)

```php
<?php
function classicray_setup() {
    add_theme_support('title-tag');
    add_theme_support('post-thumbnails');
    add_theme_support('woocommerce');
    register_nav_menus(array(
        'primary' => 'Primary Menu'
    ));
}
add_action('after_setup_theme', 'classicray_setup');

function classicray_assets() {
    wp_enqueue_style('classicray-style', get_stylesheet_uri());
}
add_action('wp_enqueue_scripts', 'classicray_assets');
```
